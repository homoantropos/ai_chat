Звісно! Для роботи з PDF у Python найчастіше використовують бібліотеку PyPDF2 або pdfplumber.

python 
import pdfplumber

def extract_text_from_pdf(pdf_path):
    with pdfplumber.open(pdf_path) as pdf:
        text = ""
        for page in pdf.pages:
            text += page.extract_text() + "\n"
    return text

pdf_text = extract_text_from_pdf("document.pdf")
print(pdf_text)

Вам потрібно буде встановити бібліотеку командою pip install pdfplumber.
